# DBpedia Dataset

Wikipedia-based text classification dataset with 14 non-overlapping classes.

## Download

Run from the repo root to download automatically (recommended):

```bash
python data/download_data.py --datasets dbpedia
```

Or manually:

1. Download `DBPEDIA_test.csv` from Kaggle:
   https://www.kaggle.com/code/danofer/dbpedia-preprocessing/input

2. Place it at:
   ```
   data/dbpedia/DBPEDIA_test.csv
   ```

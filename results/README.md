# Results

This folder contains all pipeline outputs. Files are organized by output type.

## Folder Structure

```
results/
├── clustering/                         # Clustering evaluation scores (CSV)
│   ├── benchmark/                      # Benchmark datasets (rcv1, arxiv, amazon, dbpedia, wos)
│   │   ├── {dataset}_clustering_scores.csv
│   │   ├── {dataset}_herc_*.csv
│   │   └── analysis/                   # Exploratory outputs (param sweeps, cluster assignments)
│   └── synthetic/                      # Synthetic datasets (Energy_Ecosystems, Offshore)
│       ├── {theme}_hierarchy_{config}_clustering_scores.csv
│       └── {theme}_hierarchy_{config}_herc_direct_scores.csv
│
├── viz_metrics/                        # Visualization quality metrics (CSV)
│   ├── Qwen/                           # Qwen3-Embedding-0.6B
│   │   ├── viz_metrics_{dataset}.csv              # benchmark datasets
│   │   └── viz_metrics_{theme}_hierarchy_*.csv   # synthetic datasets
│   └── sentence-transformers/          # all-MiniLM-L6-v2
│       ├── viz_metrics_{dataset}.csv
│       └── viz_metrics_{theme}_hierarchy_*.csv
│
├── summary_figures/                    # Scatter grid PNGs
│   ├── fig_scatter_grid_benchmark_{model}.png
│   └── fig2_scatter_grid_{model}.png  # synthetic scatter grids
│
├── tables/                             # LaTeX tables
│   └── {dataset}_table.tex
│
└── paper_results/                      # Figures used directly in the paper
    └── shepard_grids/                  # Shepard grid PDFs per model/config
```

## What Goes Where

| Output | Location | Generated by |
|--------|----------|--------------|
| Benchmark clustering scores | `results/clustering/benchmark/` | `eval_pipeline.py` |
| Benchmark Hercules scores | `results/clustering/benchmark/` | `herc_pipeline.py` |
| Synthetic clustering scores | `results/clustering/synthetic/` | `synthetic_eval_pipeline.py` |
| Synthetic Hercules scores | `results/clustering/synthetic/` | `synth_herc_pipeline.py` |
| Viz metrics CSVs | `results/viz_metrics/{provider}/` | `viz_metrics_script.py` |
| Scatter grids | `results/summary_figures/` | `scatter_grid_*.py` |
| LaTeX tables | `results/tables/` | manual |
| Exploratory analysis | `results/clustering/benchmark/analysis/` | manual |
| Paper figures (Shepard grids) | `results/paper_results/shepard_grids/` | manual |

## Notes

- PNG files (scatter grids, Shepard diagrams) are gitignored - not tracked in git
- CSV files in `results/` subdirectories are tracked
- `{provider}` is either `Qwen` or `sentence-transformers`

# Paper

This folder contains the paper for the NCEAS Unsupervised NLP project, written in ACL format.

The draft is located in:

```
paper/NCEAS_PHATE_Project_for_ACL/latex/
```

To compile the LaTeX:

```bash
cd paper/NCEAS_PHATE_Project_for_ACL/latex
pdflatex acl_latex.tex
bibtex acl_latex
pdflatex acl_latex.tex
pdflatex acl_latex.tex
```

The compiled PDF is `acl_latex.pdf`.
